This folder contains automation scripts created to solve real-world **operational and DevOps use cases** using shell scripting.
The scripts focus on reducing manual effort, improving reliability, and enabling scheduled execution of recurring tasks in production-like environments.
The goal of these scripts is to apply DevOps principles through **hands-on automation**, focusing on production awareness rather than theoretical examples.

**Automated Scripts:**
- System and resource monitoring
- File cleanup and maintenance automation
- AWS resource visibility and reporting
- Scheduled tasks using `cron`
- Operational utilities and helper scripts

**Tools & Technologies:**
- Bash / Shell scripting
- Linux utilities
- Cron (task scheduling)
- AWS CLI
- Git & GitHub
